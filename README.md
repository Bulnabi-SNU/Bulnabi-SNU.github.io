# Bulnabi-SNU.github.io

서울대학교 드론동아리 불나비의 홈페이지 레포입니다.



## Setup

### Environment Setup
1. ruby 설치 (윈도우는 ruby installer)
```
https://rubyinstaller.org/downloads/
```

2. jekyll, bundler 설치
```
gem install jekyll bundler
```


### Preview
레포의 루트 경로에서
```
bundle exec jekyll serve
```
위 명령 입력 시 localhost:4000에서 preview 가능
